## 🧮 Caculadora Java

Projeto desenvolvido em **Java** com o objetivo de praticar os conceitos de **Programação Orientada a Objetos (POO)**
e criação de interfaces gráficas utilizando **Java Swing**.

## 📖 Sobre o Projeto:

Esta calculadora foi desenvolvida para realizar operações matemáticas básicas por meio de uma interface gráfica simples, intuitiva e de fácil 
utilização.

Além de colocar em prática os conceitos de programação, o projeto contribuiu para o aprendizado de organização de código,
separação de responsabilidades e boas práticas em Java.

## 🚀 Funcionalidades:

➕ Soma
➖ Subtração
✖️ Multiplicação
➗ Divisão
🟰 Exibição do Resultado
 C  Limpar os Campos

## ⚒️ Tecnologias Utilizadas:
 
Java
Java Swing (Jframe)
Eclipse IDE
Git
Github

## 📂 Estrutura do Projeto:

`Model` Responsável pelos cálculos.
`view` Interface gráfica da aplicação.

## ▶️ Como Executar:

1. Clone o repositório.
2. Abra o projeto no Eclipse.
3. Execute a classe principal.
4. Utilize a calculadora normalmente.
